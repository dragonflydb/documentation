# ECHO

<PageTitle title="Redis ECHO Command (Documentation) | Dragonfly" />

## Syntax

    ECHO message

**Time complexity:** O(1)

**ACL categories:** @admin, @slow, @dangerous

Returns `message`.

## Return

[Bulk string reply](https://redis.io/docs/reference/protocol-spec/#bulk-strings)

## Examples

```shell
dragonfly> ECHO "Hello World!"
"Hello World!"
```
