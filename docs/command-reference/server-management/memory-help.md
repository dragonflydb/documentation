# MEMORY HELP

## Syntax

    MEMORY HELP 

**Time complexity:** O(1)

**ACL categories:** @slow

The `MEMORY HELP` command returns a helpful text describing the different
subcommands.

## Return

[Array reply](https://redis.io/docs/reference/protocol-spec#resp-arrays): a list of subcommands and their descriptions
